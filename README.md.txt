# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard developed using Power BI. The dashboard provides insights into employee attrition, workforce demographics, salary trends, job roles, and employee retention patterns.

## Tools & Technologies

- Power BI
- Power Query
- DAX
- CSV Dataset

## Key Performance Indicators (KPIs)

- Total Employees
- Attrition Count
- Attrition Rate
- Average Employee Age
- Average Salary

## Dashboard Features

- Department Wise Attrition Analysis
- Gender Wise Attrition Analysis
- Education Field Wise Attrition
- Job Role Wise Attrition
- Attrition by Age
- Attrition by Years at Company
- Interactive Slicers
